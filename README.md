# Transaction Risk Screening System

A Flask-based web application that simulates an early-stage transaction risk screening system using rule-based logic.

## Features
- Evaluates transactions based on amount, time, and transaction frequency
- Uses weighted risk scoring
- Classifies transactions as Approved, Requires Verification, or Blocked

## Tech Stack
- Python
- Flask
- HTML
- CSS

## Status
Prototype built for learning and demonstration purposes.
